# Design Analysis: Yellowhammer Hospitality - Courtyard on Dauphin

## Key Design Observations

### Color Palette
- **Primary**: Warm sage/taupe background (#D4B5A0 or similar warm neutral)
- **Accent**: Bright lime green (#00FF00 or similar) - used for badges, highlights, and interactive elements
- **Secondary**: Warm brown/tan (#9B8B7E or similar) for buttons and CTAs
- **Text**: Dark charcoal/black on light backgrounds, white on dark overlays
- **Contrast**: High contrast between warm neutrals and bright green creates visual interest

### Typography
- **Headings**: Bold, modern serif or sans-serif (appears to be a strong, confident font)
- **Body**: Clean, readable sans-serif
- **Emphasis**: Italics used for testimonials and quotes
- **Hierarchy**: Clear distinction between sections with varied font sizes

### Layout & Structure
- **Hero**: Full-width image with overlay text and prominent CTAs
- **Sections**: Alternating light and warm-toned backgrounds
- **Whitespace**: Generous spacing between sections
- **Asymmetry**: Images and text positioned asymmetrically for visual interest
- **Grid**: 3-column layout for feature sections

### Interactive Elements
- **Buttons**: Warm brown/tan with white text, rounded corners, lime green badges/numbers
- **Tabs**: Tabbed content sections (Description, Pricing, Co-Working Space)
- **CTAs**: Multiple prominent call-to-action buttons throughout
- **Icons**: Decorative line-drawn botanical elements (leaves, branches)

### Visual Details
- **Botanical Elements**: Hand-drawn line art of plants/leaves used as decorative accents
- **Photography**: High-quality venue photos integrated into sections
- **Badges**: Numbered lime green badges (1, 2, 3, etc.) highlighting key features
- **Dividers**: Subtle visual separators between sections
- **Shadows**: Soft shadows on cards and elements for depth

### Interaction Patterns
- **Scroll Animations**: Sections appear to have entrance animations on scroll
- **Hover States**: Interactive elements respond to hover with opacity/color changes
- **Form Focus**: Clear focus states for form inputs
- **Sticky Elements**: Navigation and CTA buttons remain accessible while scrolling

### Overall Aesthetic
- **Sophistication**: Elegant, upscale feel without being overly modern
- **Warmth**: Inviting color palette that feels welcoming
- **Grace**: Plenty of whitespace and breathing room
- **Personality**: Botanical elements and warm tones give character
- **Professionalism**: Clean layout and typography convey trustworthiness

## Key Takeaways for Redesign

1. **Color Shift**: Move from cool golds/sages to warm taupes/browns with bright lime green accents
2. **Botanical Elements**: Integrate hand-drawn plant/leaf illustrations throughout
3. **Numbered Badges**: Use bright green numbered badges to highlight key features
4. **Scroll Animations**: Add entrance animations for sections as they come into view
5. **Asymmetric Layout**: Position images and text asymmetrically for visual interest
6. **Generous Spacing**: Increase whitespace between sections for elegance
7. **Button Style**: Warm brown buttons with white text and green accent badges
8. **Photography Focus**: Integrate high-quality venue photos into sections
9. **Testimonial Styling**: Use italics and dark overlay backgrounds for testimonials
10. **Interactive Details**: Add hover effects, transitions, and micro-interactions
